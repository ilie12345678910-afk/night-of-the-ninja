🔥 Conectare Firebase Gratuită

Firebase este un serviciu Google 100% gratuit care permite sincronizarea în timp real între telefoane. Configurezi o singură dată, durează ~3 minute.

1
Deschide pe un alt tab: console.firebase.google.com → Intră cu contul Google → Add project → Dă un nume (ex: ninja) → Continue → Continue → Create
2
Din meniu stânga → Build → Realtime Database → Create Database → alege orice locație → Start in test mode → Enable
3
Copiază URL-ul care apare (arată ca: https://ninja-xxxxx-default-rtdb.firebaseio.com)

REGULILE JOCULUI
📜 Night of the Ninja

SCOP: Casa Crane 🔴 vs Casa Lotus 🔵 se luptă pentru Onoare. Primul cu 10+ puncte câștigă!

🏠 CASE ȘI RANGURI

Fiecare jucător primește o Casă secretă (Crane/Lotus + rang 1–5). Rang 1 = cel mai valoros. Supraviețuitorul cu rangul cel mai mic decide câștigătoarea rundei. Număr impar → un jucător devine Ronin (loner).

🃏 NINJA DRAFT

Primești 3 cărți Ninja. Alegi 1, pasezi 2 stânga. Primești 2 dreapta, alegi 1, arunci ultima. Rezultat: 2 cărți Ninja pe rundă.

🌙 FAZELE NOPȚII (în ordine)

🕵️ Spy: Privești Casa unui jucător.

🔮 Mystic: Privești Casa ȘI o Carte Ninja.

🃏 Trickster: Abilități speciale (schimbi case, blochezi, etc.)

🗡️ Blind Assassin: Omori fără a vedea casa!

⚔️ Shinobi: Vezi casa, apoi decizi dacă omori.

🌟 SPECIALE

🪞 Mirror Monk: Reacție — reflectă atacul înapoi!

💀 Martyr: Reacție — dacă mori, omori și tu pe cineva.

👑 Mastermind: Dacă supraviețuiești, casa ta câștigă!

🏆 ONOARE

La final de rundă, casa câștigătoare (toți membrii, vii sau morți) primesc un token de Onoare (2–4 pct). Primul la 10 puncte câștigă!
